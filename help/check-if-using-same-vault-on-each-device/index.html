How do I check if I’m using the same vault on each device?

If your 2FA accounts aren’t syncing properly across devices, it could be because each device is using a different vault. Here’s how to check:

  1. Open the Authenticator App on each device.

  2. Go to Settings > Sync & Backup.

  3. Look at the vault name shown on each device.

If the names are different, that means each device is using a separate vault, which can cause sync issues. To fix this, unify your accounts into one vault.
