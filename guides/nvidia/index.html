How to enable 2FA for your Nvidia account#

A Step-by-Step guide how to set up two-factor authentication for Nvidia

1

Download Authenticator App

  • Before we start, download the Authenticator App from your Apple device’s App Store and set it up as your main Two-Factor Authentication (2FA) option. Here is how to set up it.

    • Scan to download
2

Locate 2FA Settings

  • Sign in to your Nvidia (GeForce NOW) account, click on your account icon at the top right of the screen.

3

Security Settings

  • Under the Security section click on Update button.

4

Next Step

  • Locate Authenticator App Code and click on it.

5

Authenticator App Code

  • Click the toggle switch to turn on Authenticator App Code

6

Set up Two-Factor Authentication

  • In this step is shown the QR Code that contains the secret key which you need to scan with Authenticator App. Keep the page open as you reach for your device.
    Note: Do not scan the DEMO QR Code shown below.

7

Capture QR Code

  • Open Authenticator App, tap the plus button, and hold your device up to the computer screen to scan the QR Code from Nvidia (GeForce NOW) web page.

8

Verification code

  • After successfully scanning the QR Code given by Nvidia (GeForce NOW) on step 8, Authenticator App will automatically choose the Nvidia logo and autofill the account name and secret key. Click on Save when ready.
    Tip: You can choose the account icon by pressing on icon

9

Confirm OTP code

  • Returning to the Nvidia (GeForce NOW) web page, type in the six digit code from Authenticator App in the appropriate field. When done click OK.

10

Done!

  • You have successfully enabled two-factor authentication (2FA) to protect your Nvidia (GeForce NOW) account, from now on, you will need to use the Authenticator App when you would like to log in to your Nvidia (GeForce NOW) account.

You're all set! To learn more, visit the Nvidia support page. Stay safe!
